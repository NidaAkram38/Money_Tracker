# 💰 PocketPal – Personal Money Tracker

PocketPal is a simple and private **browser-based expense tracker** that helps you manage your **monthly budget, track expenses, and monitor spending habits**.
All data is stored locally in your browser, so your financial data remains **private and secure**.

## ✨ Features

* 🔐 **PIN Protected Access** – Secure your data with a personal PIN.
* 📊 **Monthly Budget Tracking** – Set a monthly budget and track how much you spend.
* 💸 **Expense Management** – Add, view, and delete expenses easily.
* 📅 **Automatic Monthly Organization** – Expenses are grouped by month.
* 🧾 **Expense History** – View previous months with spending summaries.
* 📂 **Category System** – Built-in categories with emojis.
* ➕ **Custom Categories** – Create your own spending categories.
* 📉 **Budget Progress Bar** – Visual indicator of how much budget is used.
* 📱 **Responsive Design** – Works on desktop and mobile devices.
* 🔒 **Private Storage** – Data is saved in **LocalStorage** (no server required).

## 🛠️ Built With

* **HTML5**
* **CSS3**
* **Vanilla JavaScript**
* **LocalStorage API**

## 🚀 How to Use

1. Open the project in your browser.
2. Create an account by setting your **name and PIN**.
3. Set your **monthly budget**.
4. Start adding expenses.
5. Track spending on the **Dashboard** and view past data in **History**.

## 🔐 Privacy

PocketTrack stores all information **locally in your browser** using LocalStorage.
No data is sent to any external server.
